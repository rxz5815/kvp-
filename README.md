# 网站导航系统

网站导航是一个简洁、可自定义的网址导航页面,基于Cloudflare Pages构建。
它允许用户轻松管理和访问常用网站链接,并支持在线编辑功能，同时还支持更换渐变色背景、随机风景图等功能。


## 功能特点

- 响应式设计,适配各种设备
- 支持在线添加和编辑链接
- 支持删除单链接与分类删除
- 密码保护的编辑功能
- 自动抓取网站图标
- 带站内搜索网站
- 带站外搜索功能
- 支持添加分类/删除分类/拖拽分类排序
- 自带一键更换背景渐变色/风景图
- 基于Cloudflare Pages,部署简单,访问快速

## 部署步骤

### 1. 克隆仓库


### 2. 创建Cloudflare Pages项目

1. 登录[Cloudflare Dashboard](https://dash.cloudflare.com/)
2. 进入"Pages"部分
3. 点击"创建项目"
4. 选择"连接到Git",然后选择您的GitHub仓库
5. 在构建设置中:
   - 构建命令: 留空
   - 构建输出目录: `/`
6. 点击"保存并部署"

### 3. 设置KV命名空间

1. 在Cloudflare Dashboard中,转到"Workers & Pages" > "KV"
2. 点击"创建命名空间"
3. 命名为"LINKS_KV"

### 4. 绑定KV到Pages项目

1. 在Pages项目设置中,转到"Functions" > "KV命名空间绑定"
2. 点击"添加绑定"
3. 变量名称设为"LINKS_KV",选择刚才创建的KV命名空间

### 5. 配置环境变量

1. 在Pages项目设置中,转到"环境变量"
2. 添加以下变量:
   - `NODE_VERSION`: 设置为`16`或更高版本
   - `EDIT_PASSWORD`: 设置为您选择的编辑密码

### 6. 重新部署

在Pages项目的"部署"选项卡中,点击"重新部署"以应用新的设置。

## 使用说明

### 访问导航页面

部署完成后,您可以通过Cloudflare Pages提供的URL或您设置的自定义域名访问导航页面。
